this repository contains the ui/ux design and interactive prototype of a mobile application for language learning, created in figma.

## project overview

the application is designed to support language learning through structured lessons and intuitive navigation.  
the focus of the project is on user experience, visual consistency, and clear user flows.

- language selection
- lesson-based learning structure
- simple and user-friendly navigation
- mobile-first design approach
